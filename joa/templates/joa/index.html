Java OceanAtlas Application

Java OceanAtlas (JOA) is a multi-OS application for viewing and exploring ocean vertical profile data. Along with a broad collection of data, JOA is the culmination of a long dream: a collection of some of the most useful oceanographic section data from the World Ocean with a multi-platform computer application for exploring those data.

Important error note (04 OCT 2017):

Due to an OS change made by Apple, it is not currently possible for Mac OS 10.12 (Sierra) users to successfully open an internet downloaded copy of the Mac OS version of Java OceanAtlas. Until a work-around is in place, Mac OS 10.12 users can obtain Java OceanAtlas only via flash drive or other file sharing mechanisms which do not involve download over the internet.i

Alternatively, see the instructions for removing Mac quarantine restrictions from downloaded software via Terminal here.
