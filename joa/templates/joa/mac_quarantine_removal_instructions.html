Java OceanAtlas Application

Mac quarantine removal instructions

Due to an OS change made by Apple, it is not currently possible for Mac OS 10.12 (Sierra) users to successfully open an internet downloaded copy of the Mac OS version of Java OceanAtlas. Until a work-around is in place, Mac OS 10.12 users can obtain Java OceanAtlas only via flash drive or other file sharing mechanisms which do not involve download over the internet.

Follow these instructions for removing Mac quarantine restrictions from downloaded software via Terminal.

Note: This workaround assumes you are somewhat comfortable using Terminal.app.

Step #1:

Enable the "Anywhere" option in Gate Keeper (Apple's name for the software that handles downloads of new software to your Mac).

This is described here:
https://howchoo.com/g/yty3mdvkyty/how-to-download-apps-from-anywhere-in-macos-sierra

Step #2:

Download JOA 5.4 from http://joa.ucsd.edu/joa

The behavior on most Macs is put an expanded zip file in my Downloads folder called "Java OceanAtlas". If you have a zip file, double click to expand into a Java OceanAtlas folder.

Step #3:

Complete Step #3 before attempting to launch JOA.

Use the terminal to navigate to your Java Oceanatlas folder in downloads using the cd command. Start in your home directory, i.e. /Users/yourusername

Issue this sequence of commands:
• cd Downloads
• cd Java\ Oceanatlas
  note: the "\" allows you to enter a space into the command
• ls
note: ls shows the contents of the folder

IMPORTANT: if you don't see:

Example Data Files Java OceanAtlas 5.app coastlines ndeditsettings.txt
JOA_Support bathymetry lexicon.txt
temp

You may not be where you think you are.

If you are not in the right folder, issue:
• cd /~yourusername and start over.

Once you are in the correct folder, issue this command:
• xattr -r -d com.apple.quarantine Java\ OceanAtlas\ 5.app

The xattr command removes a "quarantine" attribute that Apple has added to the JOA application.
